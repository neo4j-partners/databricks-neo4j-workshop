= Databricks + Neo4j: Production AI Agents with Graph and Lakehouse

== Welcome

Databricks and Neo4j solve different halves of the same problem. Databricks is the Data Intelligence Platform: it governs and analyzes structured, semi-structured, and unstructured data at scale. Neo4j is the Graph Intelligence Platform: it makes the connections between entities explicit and traversable. Databricks handles volume and governance; Neo4j handles topology and traversal. Together they form a complete intelligence platform that AI agents can query through natural language.

This workshop builds a production AI system on that pairing: a multi-agent supervisor that routes each question to the platform built to answer it, then combines the results.
