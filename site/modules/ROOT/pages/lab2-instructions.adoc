== Troubleshooting

**"Connection refused" or timeout errors** -- Verify your Neo4j URI starts with `neo4j+s://`, check your instance is running, confirm credentials are correct.

**"Spark Connector not found"** -- Ensure you are using the workshop compute (not Serverless). The compute must be in Dedicated (Single User) access mode.

**"Path does not exist" for data files** -- Verify the DATA_PATH matches your workshop configuration. Ask your instructor for the correct Volume path.

**Duplicate nodes appearing** -- The notebook uses Overwrite mode. If needed, clear your database: `MATCH (n) DETACH DELETE n`

**Notebook cells failing** -- Run cells in order from top to bottom. Do not skip the configuration cells. Check the error message for specific issues.
